# 🕸️ Lunatics Web

> **Bounded Context: Intelligence Visualization & Reporting**

The `web` workspace contains the Lunatics frontend application, a modern Angular-based dashboard for visualizing the correlation between moon phases and real-world events.

---

## 🎯 Context & Purpose

This application serves as the primary interface for users to consume the data generated by the backend pipeline. It provides historical data visualization, predictive analytics (the "Vibe"), and detailed event reports.

---

## 🏗 Domain Model

- **Dashboard:** Real-time visualization of current moon phase and crime risk.
- **Charts:** Historical trend analysis using Chart.js and ngx-charts.
- **Articles:** Reader for AI-generated reports correlating specific events with lunar cycles.

---

## 🛠 Tech Stack & Dependencies

- **Framework:** Angular 21
- **UI Components:** Angular Material, CDK
- **Data Visualization:** Chart.js, ngx-charts
- **Backend Integration:** Firebase Web SDK (Auth, Firestore, Hosting)
- **Shared Logic:** `@lunatics/shared` (Workspace dependency)

---

## 🚀 Development

### Build Commands
Run these commands from the project root using `pnpm --filter web <command>` or locally in this directory:

- **Start Dev Server:** `pnpm run start`
- **Build Production:** `pnpm run build`
- **Test:** `pnpm run test` (Vitest)
- **Lint:** `pnpm run lint`

### Configuration
Angular configuration is located in `angular.json` and build options are managed in `package.json`.
